# 🚗 Parking Management System

A simple frontend-based Parking Management System built using Agile methodology.

## ✨ Features

* Add Vehicle Entry
* Record Exit Time
* View Parking History
* Search & Summary Dashboard

## ⚙️ Tech Stack

HTML, CSS, JavaScript, Docker, Nginx

## 🐳 Run

```bash
docker build -t parking-management-system .
docker run -d -p 8081:80 parking-management-system
```

Open: [http://localhost:8081](http://localhost:8081)
